# Practice Task
## REST API application for completing generative AI tasks
Implement a standalone REST API application in python for completing generative AI tasks
Your application should be able to be run as a Docker container.
If the application connects to a database, data store etc. your project should include a docker-compose file which runs all of the required services for functional usage.

## Tasks
Text sentiment classification
Input: Text
Output: One the following classes: neutral, negative, or positive
Image description
Input: Image
Output: Text description of image contents
Text summarization
Input: text
Output: Summarization of the text

## Requirements
Developer should be able to update prompts used to perform the tasks without having to reload the application
User should be able to select a model for each tasks from a predefined list
